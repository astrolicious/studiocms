# Introduction

This Astro integration enables tracking real-world website performance and storing the data in Astro DB. That data will be used in the StudioCMS dashboard to show the performance metrics of your website from the real world over time.

## Installation

<Tabs syncKey="Installation">
  <TabItem label="Astro Method">
    <Steps>
      1. Install the package and add to your astro config using the following command:

         <PackageManagers type="exec" pkg="astro" args="add @astrojs/web-vitals" />

      2. Push the new configuration to your remote database:

         <PackageManagers type="exec" pkg="astro" args="db push" />
    </Steps>
  </TabItem>

  <TabItem label="Manual Method">
    <Steps>
      1. Install the package using the following command:

         <PackageManagers pkg="@astrojs/web-vitals" />

      2. Add `@astrojs/web-vitals` to your astro config file:

         ```js title="astro.config.mjs" ins={3, 12}
         import node from '@astrojs/node';
         import studioCMS from '@astrolicious/studiocms';
         import webVitals from '@astrojs/web-vitals';
         import { defineConfig } from 'astro/config';

         export default defineConfig({
            site: 'https://demo.astro-studiocms.xyz/',
            output: 'server',
            adapter: node({ mode: "standalone" }),
            integrations: [
                db(),
                webVitals(),
                studioCMS({
                    dbStartPage: true, // After the first time running the dev server and following the instructions at http://localhost:4321/start this will be set to false.
                }),
            ],
         });
         ```

         <PackageManagers pkg="@astrojs/web-vitals" />

      3. Push the new configuration to your remote database:

         <PackageManagers type="exec" pkg="astro" args="db push" />
    </Steps>
  </TabItem>
</Tabs>
