- **Type:** `string`
- **Default:** `en-us`

`dateLocale` is a string value that is used to determine how dates should be formatted in the `astroStudioCMS`.

## Usage

```js title="astro.config.mjs"  {2}
astroStudioCMS({
    dateLocale: 'en-us',
})
```
