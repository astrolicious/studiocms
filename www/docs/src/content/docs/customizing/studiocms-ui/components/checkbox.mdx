A custom checkbox component with support for sizes and colors. Form compatible.

## Usage

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Checkbox label="Label" color="primary" />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Checkbox } from '@studiocms/ui/components';
    ---

    <Checkbox label="Label" color="primary" />
    ```
  </TabItem>
</Tabs>

### Form Support

If you want to use this component in a form, you're in luck! There's two props you can pass: `isRequired` and `name`.

The first one does what it says it does, while the latter sets the `name` attribute so you can access the checkbox in, for example,
the `FormData` returned by a submission. Here's an example showing how to set both of them:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Checkbox label="Required & Custom Name" color="primary" isRequired name='required-checkbox' />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro ins={8-9}
    ---
    import { Checkbox } from '@studiocms/ui/components';
    ---

    <Checkbox 
      label="Required & Custom Name"
      color="primary"
      isRequired
      name='required-checkbox'
    />
    ```
  </TabItem>
</Tabs>

:::note
If no name is set, a random one is generated for the component.
:::
