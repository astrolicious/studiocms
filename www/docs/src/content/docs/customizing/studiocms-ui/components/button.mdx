import { Button } from '@studiocms/ui/components';
import { Tabs, TabItem } from '@astrojs/starlight/components';

A button component with support for different colors, sizes and states out of the box. You can also customize it to your needs!

## Usage

Using the button component is easy. Check out the example below:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Button color='primary'>
        Button
      </Button>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Button } from '@studiocms/ui/components';
    ---

    <Button color="primary">
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

### Disabled

You can disable the button component via the `disabled` attribute:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Button color='primary' disabled>
        Button
      </Button>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "disabled"
    ---
    import { Button } from '@studiocms/ui/components';
    ---

    <Button color="primary" disabled>
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

### Sizes

The button component supports three different sizes out of the box:

<Tabs>
  <TabItem label="Preview">

      </Button>
      <Button color='primary'>
        Button (default)
      </Button>
      <Button color='primary' size='lg'>
        Button (lg)
      </Button>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro /size='(?:sm|lg)'/
    ---
    import { Button } from '@studiocms/ui/components';
    ---

    <Button color='primary' size='sm'>
      Button (sm)
    </Button>
    <Button color='primary'>
      Button (default)
    </Button>
    <Button color='primary' size='lg'>
      Button (lg)
    </Button>
    ```
  </TabItem>
</Tabs>

### Polymorphic (rendered as anchor tag)

Thanks to Astro's `Polymorphic` type, you can change the button's parent component into whatever HTML tag you need it to be!
Here's an example where the button is an `<a>` tag instead of a `<button>`:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Button color='primary' target="_blank" as='a' href="https://studiocms.xyz">
        studiocms.xyz
      </Button>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "as='a'"
    ---
    import { Button } from '@studiocms/ui/components';
    ---
  
    <Button color='primary' target="_blank" as='a' href="https://studiocms.xyz">
      studiocms.xyz
    </Button>
    ```
  </TabItem>
</Tabs>

### Custom Styles

As mentioned earlier, you can apply your custom styles to the button component! Here's a gradient button to show you how it's done:

:::danger[Custom styles in production]
Due to Astro bundling CSS in an honestly, weird way, your custom styles will get overwritten in production **unless you use `!important` or make 
them inline-styles!**
:::

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Button style="background-image: linear-gradient(to top right, #a07cee, #4ff8d2); color: black; font-weight: 600;">
        Custom Button
      </Button>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code (with inline styles)">
    ```astro /style="(?:[^"]*)"/
    ---
    import { Button } from '@studiocms/ui/components';
    ---
  
    <Button style="background-image: linear-gradient(to top right, #a07cee, #4ff8d2); color: black; font-weight: 600;">
      Custom Button
    </Button>
    ```
  </TabItem>
  <TabItem label="Code (with !important)">
    ```astro ins={9-13}
    ---
    import { Button } from '@studiocms/ui/components';
    ---
  
    <Button class="custom-btn">
      Custom Button
    </Button>
    <style>
      .custom-btn {
        background-image: linear-gradient(to top right, #a07cee, #4ff8d2) !important;
        color: black !important;
        font-weight: 600 !important;
      }
    </style>
    ```
  </TabItem>
</Tabs>
