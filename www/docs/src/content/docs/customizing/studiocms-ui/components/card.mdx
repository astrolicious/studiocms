A simple card component to wrap other stuff in! 

## Usage

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card>
        Content
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Card } from '@studiocms/ui/components';
    ---

    <Card>
      Content
    </Card>
    ```
  </TabItem>
</Tabs>

### Full Width

You can set the card to fill it's parent container using the `fullWidth` param:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card fullWidth>
        Content
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Card } from '@studiocms/ui/components';
    ---

    <Card fullWidth>
      Content
    </Card>
    ```
  </TabItem>
</Tabs>

### Polymorphic

A card can be anything you want it to be, literally! Cards support Astro's `Polymorphic` type, meaning you can render the card
as whatever you want by passing the `as` attribute. Here's a card that renders as an `<a>` tag instead of a `<div>`: 

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card as="a" href="https://studiocms.xyz">
        studiocms.xyz
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Card } from '@studiocms/ui/components';
    ---

    <Card as="a" href="https://studiocms.xyz">
      studiocms.xyz
    </Card>
    ```
  </TabItem>
</Tabs>
