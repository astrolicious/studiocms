import PreviewCard from '~/components/PreviewCard.astro';

A component to display user info.

## Usage

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <User name="John Doe" description="A cool dude" />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { User } from '@studiocms/ui/components';
    ---

    <User name="John Doe" description="A cool dude" />
    ```
  </TabItem>
</Tabs>

### Avatar

You can provide an `avatar` using the same rules as the [`Image`](https://docs.astro.build/en/guides/images/#src-required) component.

import Avatar from '~/assets/avatar.png'

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <User name="John Doe" description="A cool dude" avatar={Avatar} />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "avatar={ProfileImage}"
    ---
    import { User } from '@studiocms/ui/components';
    import ProfileImage from '~/assets/profile.png'
    ---

    <User name="John Doe" description="A cool dude" avatar={ProfileImage} />
    ```
  </TabItem>
</Tabs>
