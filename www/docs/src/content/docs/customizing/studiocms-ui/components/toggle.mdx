A toggle component with support for sizes and colors. Form compatible.

## Usage

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Toggle label="Label" color="primary" />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import { Toggle } from '@studiocms/ui/components';
    ---

    <Toggle label="Label" color="primary" />
    ```
  </TabItem>
</Tabs>

### Sizes

There's three different sizes for toggles:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Row>
        <Toggle label="Small" color="primary" size="sm" defaultChecked />
        <Toggle label="Medium" color="primary" defaultChecked />
        <Toggle label="Large" color="primary" size="lg" defaultChecked />
      </Row>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro /size="[a-z]+"/
    ---
    import { Toggle } from '@studiocms/ui/components';
    ---

    <Toggle label="Small" color="primary" size="sm" defaultChecked />
    <Toggle label="Medium" color="primary" defaultChecked />
    <Toggle label="Large" color="primary" size="lg" defaultChecked />
    ```
  </TabItem>
</Tabs>

### Form Support

If you want to use this component in a form, you're in luck! There's two props you can pass: `isRequired` and `name`.

The first one does what it says it does, while the latter sets the `name` attribute so you can access the toggle in, for example,
the `FormData` returned by a submission. Here's an example showing how to set both of them:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Toggle label="Required & Custom Name" color="primary" isRequired name='example' />
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro ins={8-9}
    ---
    import { Toggle } from '@studiocms/ui/components';
    ---

    <Toggle 
      label="Required & Custom Name"
      color="primary"
      isRequired
      name='example'
    />
    ```
  </TabItem>
</Tabs>

:::note
If no name is set, a random one is generated for the component.
:::
