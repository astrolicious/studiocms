# Introduction

This Astro integration enables tracking real-world website performance and storing the data in Astro DB. That data will be used in the StudioCMS dashboard to show the performance metrics of your website from the real world over time.

## Installation

<PackageManagerTabs>
  <Fragment slot="npm">
  ```shell
  npm astro add @astrojs/web-vitals
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```shell
  pnpm astro add @astrojs/web-vitals
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```shell
  yarn astro add @astrojs/web-vitals
  ```
  </Fragment>
</PackageManagerTabs>
